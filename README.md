# git_air
TUI for git


## features 🔑

### 1. Repository Management 📁
- Create Repository (local & remote)
- Clone Repository
- Delete Repository (remote)
- Open Repository

### 2. File & Staging ⚖️
- View Staged Files
- Stage/Unstage Files
- View Diffs
- Reset

### 3. Commit Management
- Commit
- View Commit Logs
- Checkout commit
